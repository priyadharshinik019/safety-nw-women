# safety-nw-women
## 📌 Project Overview

SafeNet is a smart women’s safety application built using the MERN stack. It helps users quickly alert trusted contacts during emergency situations by sending real-time location and notifications.

The application provides a one-tap SOS feature, live GPS tracking, and instant communication to ensure faster response and improved personal safety.

## 🚀 Features

- 🔴 One-tap SOS alert with countdown  
- 📍 Live GPS tracking  
- 📲 WhatsApp alerts to trusted contacts  
- 🗺️ Google Maps live location sharing  
- 🎙️ Voice activation ("Help me")  
- 👥 Trusted contacts management  
- 📊 Alert history tracking  
- 📞 Fake call feature for unsafe situations  
- 🤝 Nearby volunteer support network
